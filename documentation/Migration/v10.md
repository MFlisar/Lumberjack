With version 10, Lumberjack dropped support for Timber as well as the viewer and notification module.

As a side effect, the structure was cleaned:

- implementations have been removed - the lumberjack implementation was integrated in the core
- loggers have been moved from "library/loggers/lumberjack" to "library/loggers"

New:

Feedback module does support iOS as well now

Changes:

* the signature of `L.sendFeedback()` has changed slightly (`fun sendFeedback(receiver: String,  appName: String, appVersion: String, attachments: List<Path>)`)
* whereever you needed to provide a mail string for feedback you now have to provide a `FeedbackConfig`
